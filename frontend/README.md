# MDGA Frontend

This is the frontend for the MDGA (Universal Data Engine) platform.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
