# 🏆 MDGA (Masterpiece Data & Governance Application)
> **2026 지역전략산업 문제해결 지식재산 리빙랩 - 최종 프로덕션 패키지**

대구·경북 지역 산업 데이터를 LLM(대형 언어 모델)과 결합하여 도시 정책과 비즈니스 전략으로 전환하는 **하이엔드 지능형 웹 앱**입니다. 현장형 리빙랩에 최적화된 모바일 퍼스트 디자인과 멀티모달 AI(Text+Vision) 엔진을 탑재하여, 지역의 문제를 데이터와 AI 기술로 즉각 해결합니다.

---

## 🎯 공모전 심사 기준 부합성 및 핵심 성과 (Core Impact)

### 1. 🚨 문제 해결의 필요성 (Problem)
*   **지역 산업 데이터의 파편화 및 디지털 소외**: 대구·경북 지역 소상공인과 중소기업의 현장 데이터가 디지털화되지 못하고 사장됨.
*   **분석 격차 (Data Divide)**: 기존 대형 상권 분석 툴은 비용이 비싸고 복잡하여 소상공인이 접근하기 어려움.

### 2. 💡 독창적 해결 방안 (Solution: MDGA)
*   **바텀업(Bottom-Up) 데이터 기여 모델**: 개별 상점의 영수증, 사진 등 비정형 데이터가 하이퍼 피딩(Hyper-Feeding)을 통해 상권, 동, 구, 대구시 전체의 '자산(Asset)'과 '활성도(Pulse)'로 즉각 집계되는 혁신적 아키텍처.
*   **리빙랩 특화 실시간 피드백 루프**: 데이터를 제공한 시민(소상공인)에게 즉각적인 AI 경영 컨설팅을 "보상"으로 제공하여 지속적인 리빙랩 참여를 유도.

### 3. 🛡️ 지식재산(IP) 및 데이터 자산화 전략 (IP Strategy)
*   **데이터 거버넌스 및 신뢰도(Trust Index) 알고리즘**: 데이터 입력 시 멀티모달 비전 검증을 통해 신뢰 지수를 산정하고, 데이터 어뷰징 적발 시 롤다운(패널티)하는 독자적 데이터 검증 엔진.
*   **공공 및 민간 융합 데이터셋 구축**: `My Store` (프라이빗) 데이터와 `Public Data` (공공 시딩)의 경계를 명확히 분리하여, 향후 지식재산권 확보 및 유료 데이터 API 사업으로의 확장 가능성 완비.

### 4. 🚀 실효성 및 기술적 완성도 (Technical Feasibility)
*   **클라우드 네이티브 & PWA 배포**: Cloudflare Pages와 Render.com 기반의 무중단 자동 배포(CI/CD) 파이프라인 구축. 스마트폰 홈 화면에 즉시 설치 가능한 PWA 적용으로 현장 사용성 극대화.
*   **실시간 동기화 (WebSocket)**: 현장의 데이터 입력이 즉각적으로 지도와 대시보드 스파크라인 차트 애니메이션으로 반영되는 라이브 관제 시스템 완비.

## 📍 플랫폼 접근 정보 (Deployment URLs)
- **프론트엔드 (Web App)**: [https://mdga-2026.pages.dev/](https://mdga-2026.pages.dev/) (스마트폰 접속 최적화)
- **백엔드 (API & WebSocket)**: `https://mdga-api.onrender.com`

## ⚙️ 시스템 인프라 및 핵심 기술 스택
- **프론트엔드**: React 19, Vite, TailwindCSS v4, Framer Motion, Lucide-React
- **백엔드**: FastAPI, Python 3.11, Uvicorn, Websockets
- **AI Core**: Google Gemini 2.5 Flash (Vision & Text Multi-modal)
- **스토리지**: 인메모리 트리 구조 (로컬 캐싱) + Google Drive API (미디어 영구 저장)

## 📂 데이터 정책 및 라이프사이클
1. **실시간 트랜잭션 데이터**: 
   - 사용자가 피딩한 데이터는 백엔드 메모리에 상주하는 **Hierarchy Engine 2.0**을 통해 관리됩니다.
   - 데이터는 `매장 -> 거리 -> 동 -> 구 -> 대구시` 형태로 상향(Roll-up) 집계되며, 모든 기기에 WebSocket으로 즉각 반영됩니다.
2. **미디어 데이터 보관**: 
   - 영수증, 매장 전경 등의 이미지 데이터는 서버 용량에 부담을 주지 않도록 **Google Drive 폴더로 자동 동기화(Upload)**되며, 앱에는 고유 `webViewLink`로 저장됩니다.
3. **가상 시뮬레이션 데이터 (초기 시딩)**: 
   - **의도**: 실 데이터를 얹기 전, 계층 구조 엔진의 안정성, AI 분석 프롬프트의 품질, 스파크라인 차트의 실시간 반응성 등을 심사위원에게 명확하게 시연하기 위한 **시스템 헬스체크 및 벤치마킹 목적의 가상 데이터셋**입니다. (공공데이터포털 연동 준비 완료)

## 🚀 데모 모드 및 심사위원 시연 가이드
1. 모바일 기기 또는 브라우저에서 [웹 앱 URL](https://mdga-2026.pages.dev/) 접속 후 페르소나 선택.
2. 대시보드 접속 후 데이터가 0개일 때 나타나는 **[✨ 데모 데이터 자동 완성 (발표용)]** 버튼을 클릭.
3. 시스템이 자동으로 고품질 데이터와 사진을 서버에 주입하며 **신뢰도, 자산 가치, 스파크라인 애니메이션이 한꺼번에 터지는 시각적 효과 확인**.
4. **[주간 리포트 발행]**을 통해 AI가 작성한 경영 리포트를 확인하고 파일로 개별 **다운로드**.
