# MDGA: Universal AI-Data Engine (Masterpiece)
> **2026 지역전략산업 문제해결 지식재산 리빙랩 - 최종 프로덕션 패키지**

## 🚀 프로젝트 개요 (Overview)
대구·경북 지역 산업 데이터를 LLM과 결합하여 도시 정책과 비즈니스 전략으로 전환하는 **하이엔드 지능형 웹 앱**입니다. 모바일 퍼스트 디자인과 멀티모달 AI(Text+Vision) 엔진을 탑재하여 현장형 리빙랩에 최적화되어 있습니다.

## 🛠️ 핵심 기술 스택 (High-Fidelity Stack)
-   **Frontend**: React (Vite) + Tailwind CSS + Framer Motion (v5.7.3)
-   **Backend**: FastAPI (Python 3.11) + Gemini 1.5 Flash Vision
-   **Storage**: Cloud Google Drive & Sheets (Real-time Integrated)
